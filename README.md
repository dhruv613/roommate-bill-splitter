## 🌸 Overview

Roommate Bill Splitter helps roommates and friends manage shared expenses without confusion.

Track:
- 🛒 Grocery bills
- 🍕 Food expenses
- ⚡ Electricity bills
- 🌐 WiFi payments
- 🏠 Rent sharing
- ☕ Random roommate spending

No more:
- “Bro, I already paid last time.”
- “Who paid the electricity bill?”
- “You still owe me ₹47.”

---

## ✨ Features

- 💰 Easy expense splitting
- 👥 Multiple roommate support
- 📊 Clean bill calculation
- 🧾 Shared expense tracking
- ❤️ Beginner-friendly project
- ⚡ Lightweight & fast
